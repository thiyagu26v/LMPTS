# Instructor Guide

Instructors track student progress and manage course enrollments.

## Viewing Student Progress

1. Open **Learner Portal**
2. Select a learner from the dropdown
3. Click **Load Progress**
4. Review enrollment table (course, status, score)

## Enrollment Management

Instructors can verify prerequisite compliance:
1. Enter learner ID and course code
2. Click **Validate** — green message means enrollment is allowed

## Completion Tracking

When a student finishes a course:
1. Enter the course code and score
2. Click **Mark Complete**
3. Progress bar and suggested courses update automatically

## Course Prerequisites

Select a course in **Admin Panel** catalog to view:
- Direct prerequisites
- Transitive (all required) prerequisites

## Analytics for Instructors

Use **Analytics Dashboard** to identify:
- Courses with low completion rates
- Bottleneck courses blocking student progress
